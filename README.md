ITMA: Investigación de Salud y Rendimiento Académico 📊
Este proyecto busca analizar la correlación entre hábitos de sueño, alimentación y el desempeño de los estudiantes del ITMA.

🛠️ Tecnologías
Python: Generación de códigos QR para recolección de datos.

HTML/CSS: Diseño de cartel informativo digital.

Data Science: (Próximamente) Análisis de resultados cuantitativos.
